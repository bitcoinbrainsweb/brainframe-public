## Decisions Made

| Decision | Chosen | Rejected alternatives |
|----------|--------|-----------------------|
| quitchat architecture | Four-skill suite | Monolith (too brittle, fails as one unit) |
| Inter-skill state passing | Each skill reads GitHub directly | Context object (single point of failure if dropped) |
| Retrospective fix delivery | Click-quiz at close | Auto-implement (unpredictable, no oversight) |
| Quality score visibility | Surfaces at close by default | Silent write only (disengagement risk) |
| Failed attempts storage | Separate persistent file | Section in handoff (not searchable across sessions) |
| GitHub write failure | Retry once then present_files | Silent drop |
| Tool call latency as friction signal | Excluded (noise) | Include (corrections/retries are better signals) |
| Perplexity re-critique on v4 | Skipped (v3 critique sufficient) | Re-run |
